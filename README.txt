VERGEX — Employee Performance Analytics Platform
================================================

HOW TO RUN:
-----------
1. Open terminal
2. cd backend
3. source venv/bin/activate  (Mac/Linux)
4. pip install flask flask-cors pandas openpyxl
5. python3 app.py

Then open: http://127.0.0.1:5500/frontend/login.html
(Use VS Code Live Server on index.html)
